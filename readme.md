# vocabulary-trainer

learn your vocabulary

## example

https://saxoneee.github.io/vocabulary-trainer/dist/

## dev

* run `npm install` once
* run `npm run dev`
* open http://localhost:8000/
* use kana or romaji (see table below)

## vocabulary

| english        | japanese kana | romaji  |
| ------------- |-------------| ----- |
| yes      | はい | hai |
| no      | いいえ | iie |
| maybe      | たぶん | tabun |
| toast      | トースト | Tōsuto (tosuto) |
